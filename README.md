UBERX — Space Travel Reimagined
UBERX es una interfaz de usuario conceptual para una aplicación de transporte interplanetario. Este proyecto explora cómo se vería y sentiría la experiencia de usuario (UX) en un futuro donde los viajes espaciales son cotidianos, combinando una estética futurista ("Space-core") con la funcionalidad de una app de movilidad moderna.

🚀 Descripción del Proyecto
El objetivo de este proyecto fue diseñar un flujo de navegación móvil coherente, desde la pantalla de bienvenida (Splash Screen) hasta la confirmación de la reserva y el recibo de pago. El diseño prioriza la inmersión visual y una jerarquía clara, utilizando una paleta de colores profunda y animaciones fluidas que evocan la tecnología espacial de vanguardia.

✨ Características Principales
Interfaz Inmersiva: Estilos CSS optimizados para dispositivos móviles (390x844px) con temas oscuros inspirados en el cosmos.

Flujo de Usuario Completo:

Splash Screen: Pantalla de entrada con branding minimalista.

Home: Panel de control principal con selección de destinos (Marte, ISS, Luna).

Reserva: Interfaz de selección de naves con visualización espacial.

Métodos de Pago: Gestión de pagos con diseño limpio y organizado.

Recibo: Visualización de detalles de transacción.

Animaciones Suaves: Implementación de transiciones personalizadas (pageEnter) para una experiencia dinámica al navegar entre pantallas.

🛠 Tecnologías Utilizadas
HTML5: Estructura semántica de las interfaces.

CSS3: Estilos avanzados incluyendo:

Variables CSS para un sistema de diseño consistente.

Flexbox para layouts responsivos.

Animaciones keyframes y cubic-bezier para transiciones fluidas.

Filtros visuales (backdrop-filter) para efectos de profundidad.

Fuentes: Integración tipográfica utilizando DM Sans (cuerpo) y Orbitron (títulos futuristas) vía Google Fonts.

📂 Estructura del Repositorio
/HTML: Archivos de marcado de cada pantalla del flujo.

/CSS: Hojas de estilo modulares que requieren una estructura de diseño global.
